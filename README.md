# Terraform 30-Day Challenge

This repository documents my journey learning **Terraform and Infrastructure as Code (IaC)** while building real infrastructure on **AWS**.

The goal of this challenge is to strengthen my skills in cloud automation, DevOps practices, and reproducible infrastructure environments.
